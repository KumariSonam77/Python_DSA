# Day 4 - Python & DSA
## Python Lists
A List is an ordered and mutable collection of elements.

### List Methods
1. append()
- Adds an element at the end of the list.

2. insert(index, value)
- Adds an element at a specific index.

3. remove(value)
- Removes an element by its value.

4. pop(index)
- Removes an element by its index.
- If no index is given, it removes the last element.

## DSA - Arrays
### Find Maximum Element
- Initialize maximum with the first element.
- Compare every element.
- Update maximum if a larger element is found.

### Find Minimum Element
- Initialize minimum with the first element.
- Compare every element.
- Update minimum if a smaller element is found.

### Find Sum of Array
- Initialize total = 0.
- Traverse the array.
- Add each element to total.

## Important Points
- List is mutable.
- Index starts from 0.
- remove() uses value.
- pop() uses index.
- append() adds at the end.
- insert() adds at a specific position.
