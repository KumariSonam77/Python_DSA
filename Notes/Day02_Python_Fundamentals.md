#Day 2 Notes - Python Fundamentals, Loops & DSA Basics
## Objective
Learn Python fundamentals, understand loops, practice basic pattern programming, and study the basic concepts of DSA.
---
# 1. Variables
A variable is a named container used to store data in a program.

*Example:*
name = "Sonam"
age = 20
---

# 2. Data Types
A **data type** defines the type of value stored in a variable.
| Data Type | Description     | Example   |
| --------- | --------------- | --------- |
| int       | Whole numbers   | `10`      |
| float     | Decimal numbers | `3.14`    |
| str       | Text            | `"Hello"` |
| bool      | True or False   | `True`    |
---

# 3. Input
The `input()` function is used to take input from the user.
*Example:*
name = input("Enter your name: ")
---

# 4. Output
The `print()` function is used to display output on the screen.
*Example:*
print("Hello")
---

# 5. Type Casting
Type casting converts one data type into another.
*Example:*
age = int(input("Enter age: "))
---

# 6. Operators
Operators perform operations on values or variables.
# Arithmetic Operators

* `+` Addition
* `-` Subtraction
* `*` Multiplication
* `/` Division
* `%` Modulus (Remainder)

### Comparison Operators
* `==` Equal to
* `!=` Not equal to
* `>` Greater than
* `<` Less than
* `>=` Greater than or equal to
* `<=` Less than or equal to
These operators return either **True** or **False**.

# 7. if-else Statement
The `if-else` statement is used to make decisions based on a condition.
*Example:*
if age >= 18:
    print("Eligible")
else:
    print("Not Eligible")
---

# 8. Loops
A **loop** is used to execute a block of code repeatedly.
## For Loop
Used when the number of iterations is known.
*Example:*
for i in range(5):
    print(i)
```
## While Loop
Used when execution depends on a condition.
*Example:*
while i <= 5:
    print(i)

### Difference
* **For Loop:** Used when the number of repetitions is known.
* **While Loop:** Used when repetition depends on a condition.

# 10. Algorithm
An **algorithm** is a step-by-step procedure to solve a problem.
**Example:**
1. Start
2. Take Input
3. Process Data
4. Display Output
5. End
--

# 11. Pseudocode
**Pseudocode** is a simple, language-independent way of writing the logic of a program before coding.
**Example:**
Start
Take input
Process data
Display result
End
---

# 12. Flowchart
A **flowchart** is a graphical representation of an algorithm.
Basic flow:

Start
  ↓
Input
  ↓
Process
  ↓
Decision
  ↓
Output
  ↓
End
